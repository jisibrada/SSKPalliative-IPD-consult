✏️ แก้ไขข้อมูลผู้ป่วย

🔗 เชื่อมต่อ Google Sheets (Two-way Sync)

ระบบนี้รองรับ sync สองทิศทาง — เว็บส่งข้อมูลเข้า Sheet และดึงข้อมูลจาก Google Form มาแสดงใน Dashboard

  1. เปิด Google Sheet ที่ Google Form ของคุณเชื่อมอยู่แล้ว
    (Form → Responses → คลิกไอคอน Spreadsheet)
  2. ไปที่ Extensions → Apps Script → ลบ code เดิมออกทั้งหมด
  3. วาง code จากไฟล์ palliative_apps_script.gs ที่ได้รับ
  4. ตรวจสอบชื่อ sheet ใน code บรรทัด FORM_SHEET_NAME ให้ตรงกับชื่อ tab ใน Spreadsheet
  5. กด Deploy → New deployment → Web App
    Execute as: Me · Who has access: Anyone
  6. Copy Web App URL มาวางด้านล่าง แล้วกดบันทึก
ทดสอบก่อน Deploy
ใน Apps Script ให้รัน function testSetup() เพื่อตรวจสอบว่า script เจอ Sheet ถูกต้อง ดูผลได้ที่ Execution log
หลังเชื่อมแล้ว: ไปที่ Tab Dashboard → กด "ดึงข้อมูลจาก Sheets" เพื่อโหลดข้อมูลจาก Google Form และ Web Consult Sheet พร้อมกัน
